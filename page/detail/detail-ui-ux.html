Tentang Program

Dapatkan Kelas Gratis UI/UX Design

Bersama DigiSkill anda bisa mendaftar kelas UI/UX Design dengan gratis tanpa
dipunggut biaya sedikitpun.

Bidang UI/UX sangat diminati oleh dunia industri belakangan ini. UI/UX dapat membantu menciptakan website atau aplikasi
yang memenuhi kebutuhan user dan meminimalisir ketidaknyamanan user dalam menggunakan suatu website ataupun
aplikasi mobile.

Dengan arahan mentor, kali ini kita akan belajar bagaimana cara membuat portfolio website dengan memanfaatkan animasi
UI serta memastikan kualitas interface dan experience project berjalan baik menggunakan tools Figma.

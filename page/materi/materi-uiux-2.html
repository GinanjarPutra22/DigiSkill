Apa saja basic tools yang ada pada Figma?

Figma adalah aplikasi desain berbasis cloud dan alat prototyping untuk project digital. Figma dapat membantu para penggunanya berkolaborasi dalam suatu project atau pekerjaan dengan banyak pengguna secara real-time dan di mana saja.

Figma juga merupakan aplikasi desain UI dan UX yang dapat kamu gunakan untuk membuat situs web, aplikasi, atau komponen antarmuka pengguna.

Dengan arahan website, kali ini kita akan belajar mengenai basic tools pada Figma dengan melihat video yang sudah disiapkan.
